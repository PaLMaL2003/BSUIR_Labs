Основы веб-разработки с HTML

На этом занятии студенты изучат основы языка разметки HTML. Они научатся создавать структуру веб-страниц, использовать теги для разметки текста, вставлять изображения и ссылки. Кроме того, студенты познакомятся с основными элементами форм и научатся правильно организовывать контент на странице с помощью HTML.

Стилизация веб-страниц с помощью CSS

На этом занятии студенты узнают, как использовать CSS для стилизации веб-страниц. Они изучат основные понятия каскадных таблиц стилей, такие как селекторы, свойства и значения. Студенты научатся изменять цвета, шрифты, размеры и расположение элементов на странице с помощью CSS. Кроме того, они познакомятся с адаптивным дизайном и медиазапросами.

Взаимодействие с веб-страницами через JavaScript

На этом занятии студенты будут изучать язык программирования JavaScript и его возможности для взаимодействия с веб-страницами. Они узнают, как добавлять интерактивность к страницам, обрабатывать события пользователя, изменять содержимое и стили элементов, выполнять валидацию форм и создавать простые анимации. Студенты также познакомятся с основами работы с DOM (объектной моделью документа) и изучат основные концепции программирования на JavaScript.
